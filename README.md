# Pomodoro Timer

A modern, web-based Pomodoro timer application built with React and Vite. Focus on your tasks with a beautiful, distraction-free interface.

## Features

- **Custom Timer**: Set your preferred focus duration (default: 25 minutes).
- **Controls**: Start, Pause, and Reset functionality.
- **Visuals**: Modern "glassmorphism" design with ambient lighting and dark mode aesthetics.
- **Audio Notifications**: Gentle beep notification when the timer completes.
- **Responsive**: Works brilliantly on desktop and mobile.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1.  Clone the repository or navigate to the project directory.
2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Start the development server:

    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:5173`.
